# About The Project

<WorkInProgress />

Whendarr is an open source, self hosted calendar that aggregates release data from *arr services into a simple monthly interface.
Whendarr aims to remove the need to expose tools like Sonarr and Radarr to end users. Instead, providing a fast, shareable, readonly view of upcoming show and movie releases.
Aggregation also includes availability statuses and metadata, so that your media stack can stay private, but users can stay informed, and possibly find new content to explore.

## Key Features

### Media Integration

- Integration with Sonarr and Radarr
- Aggregated Show and Movie release in a unified calendar view
- Automatic grouping of episodes by show for cleaner navigation

### Calendar Experience

- Clear visibility into upcoming releases
- Download/availability status indicators for media
- Dark and Light theme support
- Mimimal UI focused on usability

### Privacy & Security

- Keep your *arr stack private, no need to expose Sonarr or Radarr
- Read-only, shareable interface

### Performance & Infrastructure

- Environment variable configuration support (including `_FILE` for secrets)
- Redis/Valkey caching to improve load time and reduce load on *arr stack
- Designed for easy reverse proxy and subpath setups

## Planned Features

- Filtering and personalized views
- Planned timezone for accurate scheduling
- Planned internationalization
