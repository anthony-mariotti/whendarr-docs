# Environment Variables

<WorkInProgress />

Whendarr uses environment variables for complete configuration.

## Required Environment Variables

| Variable | Description | Example |
|:---------|:------------|:--------|
| `RADARR_URL` | Radarr Instance URL | `http://radarr.domain.tld` |
| `RADARR_KEY`<br/>`RADARR_KEY_FILE` | Radarr API Key | `867530900000000000` |
| `SONARR_URL` | Sonarr Instance URL | `http://sonarr.domain.tld` |
| `SONARR_KEY`<br/>`SONARR_KEY_FILE` | Sonarr API Key | `867530900000000000` |

## Optional Environment Variables

### Web Server / Hosting

| Variable | Description | Default |
|:---------|:------------|:--------|
| `BASE_PATH` | Use a path to host under | `/` |
| `HOST` | Bind to a specific host interface | `0.0.0.0` |
| `PORT` | Bind to a specific port | `3000` |
| `LOG_LEVEL` | `fatal` \| `error` \| `warn` \| `info` \| `debug` \| `trace` | `info` |
| `TRUSTED_PROXY` | Trust a specific proxy `IP` address | Not Set |
| `TRUSTED_PROXY_HOP` | Trust a specific number of reverse proxies | Not Set |

### Redis/Valkey Caching

| Variable | Description | Default |
|:---------|:------------|:--------|
| `REDIS_URL`\*<br/>`REDIS_URL_FILE`\* | Redis connection string | Not Set |
| `REDIS_HOST` | Redis instance host/ip | Not Set |
| `REDIS_PORT` | Redis instance port | `6379` |
| `REDIS_USERNAME`<br/>`REDIS_USERNAME_FILE` | Redis instance username | Not Set |
| `REDIS_PASSWORD`<br/>`REDIS_PASSWORD_FILE` | Redis instance password | Not Set |
| `REDIS_PREFIX` | Redis key prefix for namespacing | `whendarr` |
| `REDIS_TLS` | Enable the use of TLS | `false` |
| `REDIS_TLS_CA`<br/>`REDIS_TLS_CA_FILE` | TLS trusted CA certificate | Not Set |
| `REDIS_TLS_CERT`<br/>`REDIS_TLS_CERT_FILE` | TLS certificate file | Not Set |
| `REDIS_TLS_KEY`<br/>`REDIS_TLS_KEY_FILE` | TLS certificate key | Not Set |
| `REDIS_TLS_REJECT_UNAUTHORIZED` | Redis key prefix for namespacing | `true` |

\* `REDIS_URL` takes priority over `REDIS_HOST`, `REDIS_PORT`, `REDIS_USERNAME`, and `REDIS_PASSWORD`
