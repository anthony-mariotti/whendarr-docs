# Docker

<WorkInProgress />

## Command Line

```bash
docker run --name whendarr \
    -e RADARR_URL=value \
    -e RADARR_KEY=value \
    -e SONARR_URL=value \
    -e SONARR_KEY=value \
    -p 3000:3000 \
    ghcr.io/anthony-mariotti/whendarr:latest
```


## Compose File

```yml
services:
  whendarr:
    image: ghcr.io/anthony-mariotti/whendarr:latest
    environment:
      RADARR_URL: <radarr_instance>
      RADARR_KEY: <radarr_apikey>
      SONARR_URL: <sonarr_instance>
      SONARR_KEY: <sonarr_apikey>
    ports:
      - 3000:3000
```
